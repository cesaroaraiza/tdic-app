📦 Parts tracker setup

Connect to a free Supabase database so your whole team shares the same data. Do this once — after that everyone just opens the URL.

1
Go to supabase.com, create a free account, and start a new project.
2
Open the SQL Editor and run this to create both tables:

create table movements (id bigserial primary key, type text, part text, qty int, from_loc text, to_loc text, ref text, source_type text, notes text, logged_by text, job_number text, client_name text, created_at timestamptz default now());

create table settings (key text primary key, value text);
3
Go to Project Settings → API and copy your Project URL and anon public key.
4
Paste them below and click Connect. Share this HTML file with your team.
